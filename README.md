# Appendix for modelcomp

[modelcomp](https://github.com/gilramot/modelcomp) is a package developed for a microbiome research. This repo contains the files essential for reproducing the results of the research.

## Files

[*main.py*](https://github.com/gilramot/modelcomp-appendix/blob/main/main.py) - the module that was used for the research. Requires the first release of modelcomp installed.

[*base-article*](https://github.com/gilramot/modelcomp-appendix/tree/main/base-article) - the base article for the research (uses its dataset)

[*data*](https://github.com/gilramot/modelcomp-appendix/tree/main/data) - the input dataset ([*abundance.csv*](https://github.com/gilramot/modelcomp-appendix/blob/main/data/abundance.csv) stores the microbiomial data, [*meta.csv*](https://github.com/gilramot/modelcomp-appendix/blob/main/data/meta.csv) stores the target variable)

[*export*](https://github.com/gilramot/modelcomp-appendix/tree/main/export) - the output of main.py

[*disease-keys.txt*](https://github.com/gilramot/modelcomp-appendix/blob/main/disease-keys.txt) - the possible values of the target variable
